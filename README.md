# bitcoin_cleaner

El presente script fue desarrollado por Rafael Reyna Camones para el curso de Tipología y ciclo de vida de los datos.
Analiza un dataset conteniendo información acerca de 200 criptmonedas, obteniendo las cinco con mayor valor de capitalizacion.
Por cada una de estas monedas busca determinar el dia que fue más rentable comprar dicha moneda durante el 2017.

Para ejecutar el script es necesario instalar la siguientes bibliotecas:
```
pip install pandas
pip install scipy
pip install matplotlib
pip install numpy
```
